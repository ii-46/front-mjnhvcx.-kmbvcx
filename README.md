# front-inventory

## 💿 Install

Use your selected package manager (yarn) to install dependencies:

```bash
yarn install
```

## 🚀 Quick Start

```bash
yarn install
yarn dev
```

## 🏗️ Build

```bash
yarn build
```

## 🧪 Available Scripts

- `yarn dev`
- `yarn build`
- `yarn preview`
- `yarn build-only`
- `yarn type-check`
